⚡ XAUUSD ML Bot

Торговый бот на основе машинного обучения для золота

XAUUSD / Gold LightGBM · Direction-Only M1 · BUY + SELL Level Bot · Уровни Metka Bot · Паттерны Swing Bot · $4 импульс TP/SL — только в боте
🔑 Ключевое изменение архитектуры: все модели теперь предсказывают только направление цены (вверх / вниз) — без фиксированных TP/SL в метке при обучении. TP и SL задаются исключительно при запуске бота (--tp-points, --sl-points). Обученная модель работает с любой комбинацией TP/SL — переобучение не нужно.

Metka Bot — бот на основе логики индикатора metka.mq5. Торгует по тем же паттернам что рисует индикатор на графике (пин-бар, разворот импульса, squeeze breakout, RSI-дивергенция). Паттерны Метки (дефолтные настройки)

Бот вычисляет те же сигналы что индикатор Metka v28 с параметрами по умолчанию. Выключены: объём, ADX, Bollinger. Включены: Swing-фильтр.

ПаттернУсловие BUYПризнак
Пин-барНижняя тень >50% диапазона, диапазон >2×ATR, закрытие выше серединыpin_bar_buy
Разворот импульса≥3 медвежьих баров подряд + сильный бычий бар (тело >ATR×0.3)momentum_rev_buy
Squeeze BreakoutСжатие диапазона (<0.5 ATR) + пробой вверх + EMA21 > EMA50squeeze_break_buy
RSI ДивергенцияЦена у нового минимума, RSI выше предыдущего минимума на 3+rsi_div_buy

Все паттерны дополнительно проходят Swing-фильтр: BUY только если цена у локального минимума (окно 15 баров, допуск 1.5 ATR).

Итоговый сигнал: metka_buy / metka_sell — любой паттерн + swing-фильтр.

metka_buy_strength — сколько паттернов сработало одновременно (0–4). Чем выше, тем сильнее сигнал.

EMA Fast=21, EMA Slow=50 — настраиваются в UI Metka (input-параметры).
ATR=20, RSI=14 — жёстко зашиты в код Metka как внутренние константы (не видны в настройках индикатора).

Построить датасет с Metka-признаками

Метка строится по двум условиям (OR) за 8 баров:

УсловиеLongShort
1 — итог через 8 баровclose[i+8] > close[i]close[i+8] < close[i]
2 — касание TP до 8-го бараmax(high) >= close[i] + $0.80min(low) <= close[i] − $0.80

label = 1 если выполнено хотя бы одно. Это позволяет модели видеть быстрые контртрендовые сделки которые дали TP за 1–2 бара и не потерялись при взгляде только на бар 8.

ПараметрПо умолчаниюОписание

1

Концепция и математика

Модель обучается предсказывать направление цены через 20 минут. TP и SL задаются при запуске и не влияют на качество модели:

Условие меткиЗначение
close[i+20] > close[i]1 — цена выросла (long)
close[i+20] < close[i]0 — цена упала

При winrate ~55–60% на хорошей модели и R:R = 4:1 (400 TP / 100 SL) математика сходится:

Breakeven при R:R 4:1: нужно > 20% winrate (100 / (400+100) = 20%)
При WR=55%:  PnL = 0.55 × 400 − 0.45 × 100 = 220 − 45 = +175 пунктов / сделку
При WR=28%:  PnL = 0.28 × 400 − 0.72 × 100 = 112 − 72 = +40 пунктов / сделку
Прибыль зависит от лота: 0.1 лот → $1 = 100 пунктов
При R:R 4:1 и лоте 1.0: TP=$4.00/сделку (400 пунктов × $0.01)
